About Bricks

Bricks is a visual application development tool that makes it easy for anyone to build database-driven web and mobile applications in minutes.

The applications generated with Bricks use Bootstrap for the interface and AngularJS for the dynamic components. They can be hosted on any web server or wrapped with PhoneGap (or similar tools) for deployment on mobile devices.

For more information, please go to the Github repository.
